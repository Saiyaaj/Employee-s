# EMS React App

This is a React + Vite employee management system (EMS) demo application.

## Features

- Admin login and employee login support
- Admin dashboard for task creation and team assignment
- Employee dashboard with task cards
- Interactive task actions: Accept, Complete, Fail
- Local storage persistence for demo users and tasks
- Tailwind CSS styling and responsive UI

## Setup

```bash
npm install
npm run dev
```

Open the local `vite` URL shown in the terminal.

## Build

```bash
npm run build
```

## Notes

- The app uses browser `localStorage` for seeded users and tasks.
- Changes made by employees are persisted only in the browser storage.
